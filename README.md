# canton-node-sdk

Low-level TypeScript SDK for Canton Network nodes (Ledger JSON API, Validator API, Scan API).

- **Documentation:** [GitHub wiki](https://github.com/fairmint/canton-node-sdk/wiki) — architecture, LocalNet, examples, and deep dives
- **Hosted docs:** [canton-node-sdk.fairmint.com](https://canton-node-sdk.fairmint.com/) (built from [fairmint/web](https://github.com/fairmint/web), `apps/canton-node-sdk`)
- **Install:** `npm install @fairmint/canton-node-sdk`

Contributor setup: [CONTRIBUTING.md](https://github.com/fairmint/canton-node-sdk/blob/main/CONTRIBUTING.md).
