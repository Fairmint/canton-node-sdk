# canton-node-sdk

## Step 0: Read the Wiki (REQUIRED)

All documentation lives in the [GitHub wiki](https://github.com/fairmint/canton-node-sdk/wiki). Before doing
any work in this repository, clone or update the local copy and read `Home.md`:

```bash
if [ -d wiki/.git ]; then git -C wiki pull; else git clone https://github.com/fairmint/canton-node-sdk.wiki.git wiki; fi
```
