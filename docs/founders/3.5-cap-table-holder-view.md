# Cap Table Holder View

*This article is for founders and admins using Fairmint Studio.*

The Holder View shows your cap table **grouped by stakeholder** — each person or entity, what they hold, and their ownership percentage.

Go to **Cap Table > Holder View** in the left menu.

For the class-by-class breakdown, see [Summary View](3.4-cap-table-summary-view.md).

---

## Before Your Cap Table Is Ready

Same as Summary View — if your cap table hasn't been set up or has no data, you'll see a setup prompt. Go to [Timeline View](3.6-cap-table-timeline-view.md) to upload documents, or see [Import Your Cap Table](3.1-upload-import-cap-table.md).

---

## Toolbar

The Holder View shares the same toolbar as Summary View:

- **Today / As-Of Date** — view your cap table as of any past date
- **Pro-Forma** — model hypothetical scenarios (financing round, convert SAFEs/notes, option pool)
- **Export** — download as a spreadsheet (includes Pro-Forma comparison if active)

As-Of and Pro-Forma are mutually exclusive. See [Summary View](3.4-cap-table-summary-view.md) for detailed descriptions of each toolbar feature.

---

## Validation Warnings

Same as Summary View — if data issues are detected, a yellow warning banner appears at the top listing the specific problems.

---

## What You See

The page heading shows **"Holder View"** and the table has two tabs: **Holder View** and **Voting Rights**.

### Holder View Tab

The default tab shows a table with columns:

| Column | Description |
|--------|-------------|
| STAKEHOLDER | Name and relationship (Founder, Employee, Investor, etc.) |
| TOTAL STOCK (OUTSTANDING)* | Shares currently issued and outstanding |
| TOTAL STOCK (AS CONVERTED) | Shares including conversion of convertibles |
| TOTAL STOCK % (AS CONVERTED) | Percentage of total as-converted shares |
| FULLY DILUTED SHARES** | All shares including options, warrants, and convertibles |
| FULLY DILUTED % | Percentage of total fully diluted shares |

Each stakeholder appears as a row with an expand arrow (▼) to see details of their individual securities. Stakeholders are sorted by ownership size, with the most significant holders first.

At the bottom of the table:

- **Options Available for Issuance** — shows the unallocated option pool (from your Stock Plan)
- **TOTAL** — sums all outstanding and fully diluted shares to 100%

### Voting Rights Tab

The **Voting Rights** tab shows the voting power distribution across stakeholders, reflecting any share classes with different voting weights.

---

## Pro-Forma and As-Of in Holder View

Both Pro-Forma and As-Of data carry over from Summary View — if you set an As-Of date or generate a Pro-Forma scenario in Summary View, switching to Holder View preserves that context. The same banners appear, and the same data is reflected.
