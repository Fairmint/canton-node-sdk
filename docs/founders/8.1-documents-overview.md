# Documents Overview

*This article is for founders and admins using Fairmint Studio.*

The Documents page is your central file repository — all documents generated or uploaded through Fairmint are organized here. Click **Documents** in the left menu to open it.

## What You'll See

Documents are organized into folders. Each folder shows a file count on the right (e.g., "88 files"). Click any folder to expand it and see individual files.

### Standard Folders

These folders appear at the top of the page. You'll only see a folder when it contains at least one file.

| Folder | What It Contains |
|--------|-----------------|
| **Uploaded** | Cap table documents and custom legal documents you've uploaded |
| **Shareholder Registers** | PDF reports generated from your cap table (see [Generate a Shareholder Register PDF](8.2-generate-shareholder-register-pdf.md)) |
| **Cap Table Exports** | Exported cap table spreadsheets |
| **Company Documents** | Company-level documents |
| **KYC** | Identity verification documents from investors |
| **Accreditation Proof** | Accreditation verification documents from investors |

### Series Folders

Below the standard folders, each fundraising or grant series gets its own folder (listed in alphabetical order). These contain the signed agreements and documents related to that series — investment agreements, grant agreements, and any custom legal documents.

## Actions

### Download Documents

Click **Download Documents** at the top of the page to download all your documents at once as a ZIP file. On smaller screens, the button is labeled **Download**.

### Generate Report

Click **Generate Report** to create a shareholder register PDF from your cap table data. On smaller screens, the button is labeled **Report**. This button appears once your cap table has been set up. See [Generate a Shareholder Register PDF](8.2-generate-shareholder-register-pdf.md) for details.

## When You're Just Getting Started

If you haven't created any series or uploaded any documents yet, you'll see:

> "📁 Your files will appear here once you've begun using Fairmint. Examples include cap table and fundraising documents."
