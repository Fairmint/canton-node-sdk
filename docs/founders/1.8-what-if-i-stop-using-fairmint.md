# What if I Want to Stop Using Fairmint, or Fairmint is Bought or Goes Out of Business?

*This article is for both founders and investors using Fairmint.*

Subject to the terms in any applicable agreement, you are free to stop using Fairmint at any time. Users can pause their offerings for temporary stoppage, or cancel their Fairmint portal entirely by following the cancellation procedures outlined in the applicable agreement.

Cancelling your agreement will only cancel your Fairmint software access and license, and will not cancel or void any smart contracts or digital assets deployed or issued while using Fairmint software.

The smart contracts and digital assets deployed and issued while using Fairmint software are deployed to the Canton Network, meaning they exist and function independently of Fairmint. These technologies and assets will continue to exist and operate if you chose to stop using Fairmint, as long as the Canton Network is maintained and active, and subject to a loss in certain functionalities or features if Fairmint software is no longer being used.
