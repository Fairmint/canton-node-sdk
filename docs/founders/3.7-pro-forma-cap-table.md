# Pro-Forma Cap Table

*This article is for founders and admins using Fairmint Studio.*

Pro-Forma mode lets you model hypothetical scenarios — like a new funding round, converting your SAFEs, or expanding your option pool — and instantly see how they would affect ownership on your cap table. No data is saved until you decide to make real changes.

Go to **Cap Table > Summary View** and click the **Pro-Forma** button in the top-right corner.

> **Note:** Pro-Forma is not available while you're viewing a historical "As Of" date. Exit the As Of view first.

---

## Scenario Types

When you click **Pro-Forma**, a dialog opens with three scenario types. Select one, fill in the fields, and click **Generate Pro-Forma**.

### Financing Round

Model a new equity investment to see how it dilutes existing shareholders.

| Field | Required | Description |
|-------|----------|-------------|
| **Pre-money Valuation** | Yes | Your company's valuation before the investment (e.g., $10,000,000) |
| **Investment Amount** | Yes | The amount being invested (e.g., $2,000,000) |
| **Stock Class Name** | Yes | Name for the new share class (e.g., "Series A Preferred") |
| **Lead Investor Name** | No | Name of the lead investor (optional) |
| **Target Pool Percentage** | No | If you want to expand your option pool as part of this round, enter the target percentage of post-money fully diluted shares. Leave empty to skip pool expansion |

If you enter an option pool percentage, you'll also choose the **Pool Expansion Timing**:

- **Pre-money (typical)** — the pool dilutes existing shareholders only, not the new investor
- **Post-money** — the pool dilutes everyone, including the new investor

### Convert SAFEs / Notes

See what happens when all outstanding SAFEs and convertible notes convert into equity at a given valuation.

| Field | Required | Description |
|-------|----------|-------------|
| **Trigger Valuation** | Yes | The valuation that triggers conversion — each SAFE/note converts using its own cap and discount terms |

All outstanding SAFEs and Notes are included automatically.

### Option Pool

Model expanding your stock option pool to a target size.

| Field | Required | Description |
|-------|----------|-------------|
| **Target Pool Percentage** | Yes | The target percentage of fully diluted shares you want the pool to reach |

New shares are added to bring the pool up to the target percentage.

## Reading the Results

After clicking **Generate Pro-Forma**, the page switches to Pro-Forma mode:

### Banner

A yellow banner appears above the cap table with:

- A **"Pro-Forma Mode"** label with a tooltip reminding you this is hypothetical — no data has been saved
- **Key metrics** for your scenario (varies by type):
  - Financing Round: Investment, Price/Share, New Shares, Post-Money, Pool Expansion (if applicable)
  - Convert SAFEs/Notes: Trigger Valuation, Total Converted, New Shares, number of Conversions
  - Option Pool: Shares Added, New Pool Size, New Pool %
- **Dilution impact** — if existing holders are diluted, you'll see a line like: *"Impact: 500,000 new shares (10.5% dilution to existing holders)"*
- **Errors or warnings** — if the computation encounters issues (e.g., missing data), they appear as expandable alerts in the banner

### Cap Table

The Summary and Holder views update to reflect the pro-forma scenario. The table layout stays the same — same columns, same groupings — but the numbers reflect the hypothetical outcome. You can switch between Summary and Holder views while in Pro-Forma mode.

### Export

The **Export** button turns yellow while Pro-Forma is active. Clicking it exports the pro-forma cap table (including comparison data) so you can share the scenario with your team or advisors.

---

## Editing or Exiting Pro-Forma

While in Pro-Forma mode, the header button changes to **"Edit Pro-Forma"** with an **X** next to it:

- **Edit Pro-Forma** — reopens the dialog with your current inputs so you can adjust the scenario
- **X (Exit)** — clears the pro-forma and returns to your actual cap table

Pro-Forma mode is also cleared if you switch to a different company or apply an "As Of" historical date.
