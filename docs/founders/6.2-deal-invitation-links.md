# Use Deal Invitation Links

*This article is for founders and admins using Fairmint Studio.*

Invitation links are unique URLs you share with investors to give them access to your fundraise. You can create multiple links with different investment limits and verification settings for different investor groups.

## Create an Invitation Link

1. From your series detail page, click **Create new invitation link** (with a + icon) in the top-right area
2. On the **Create an invitation link** form, fill in:
   - **Name** — a label to identify this link (e.g., "Angels", "VC funds", "Friends & family")
   - **For VCs and Qualified Investors Only** — check this to fast-track qualified investors (for private, published series)
   - **Set a fixed amount** — check this to require a specific investment amount, or leave unchecked to set a range
   - **Minimum amount** — the smallest investment you'll accept (defaults to $2,500; minimum allowed is $500)
   - **Maximum amount (optional)** — the largest investment allowed. Leave blank for no upper limit
   - **Require manual validation for investors using this link.** — if checked, investors must be approved by you before they can invest
   - **Require ID verification for investors using this link.** — if checked, investors must verify their identity
   - **Send and restrict invitation link to:** — optionally restrict the link to specific email addresses. When you enter an email (e.g., partner@sequoia.com), a new option appears: **"Allow anyone from sequoia.com to access this link"**. Check this to let anyone with a verified email from that domain invest or review the deal — useful for VC firms where multiple people need access. This option is not available for generic email providers (Gmail, Yahoo, etc.).
3. Review the **Investor preview** panel on the right side to see what investors will experience — enter a total amount to preview the investment summary (investment amount, processing fees, total)
4. Click **Create link**

**Important:** If neither verification checkbox is selected, a warning explains that all investors using this link will be treated as individuals you personally know and trust. They won't go through identity verification. To enable investor verification, you need a Fairmint One membership.

**Note:** On the Free plan, the manual validation and ID verification controls are disabled. Upgrade to Fairmint One to access these features.

If the series has a **token warrant**, the warrant purchase price is also shown on the form (read-only).

## Manage Your Links

Go to the **Invitation Links** tab on your series detail page to see all your links.

### Before Publishing (Setup / Ready)

The table shows:

| Column | Description |
|--------|-------------|
| **INVITATION LINK** | Link name and URL path (e.g., `/invitation/f1c7nc6lucal`) with a copy icon |
| **INVESTMENT LIMITS** | Min/max amounts configured |
| **MANUAL VALIDATION** | Whether manual approval is required |
| **ID VERIFICATION** | Whether ID check is required |
| **EDIT** | Pencil icon to edit settings, trash icon to delete the link |

### After Publishing (Live)

The table changes:

| Column | Description |
|--------|-------------|
| **INVITATION LINK** | Link name and URL with copy icon |
| **INVESTMENT LIMITS** | Min/max amounts (click pencil icon to edit) |
| **MANUAL VALIDATION** | Whether manual approval is required |
| **ID VERIFICATION** | Whether ID check is required |
| **ACTIVE** | Switch to enable or disable the link |

For non-private series, a **Public page** row also appears at the top with the public deal page URL.

## Copy and Share Links

Each link shows its URL path with a **copy icon** next to it. Click the copy icon to copy the full invitation URL to your clipboard, then share it with investors via email, messaging, or any other channel.

**Note:** Your custom domain must be set up for invitation links to work. If it's not configured, you'll see a warning with a link to complete the setup.
