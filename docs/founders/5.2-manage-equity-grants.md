# Manage Equity Grants

*This article is for founders and admins using Fairmint Studio.*

Once you've created a grant series (see [Set Up an Equity Grant](5.1-set-up-an-equity-grant.md)), this article covers how to manage it — from the grant series detail page to tracking individual grants through their lifecycle.

## Grant Series Detail Page

Click a grant series in the **Grants** section of the left menu to open its detail page. The page contains:

### Header & Actions

The header shows the series name with an edit icon and a **live** label (once published). Two buttons appear in the top right:

- **+ New equity grant** — create a new individual grant from this series
- **Preview** — see what grant recipients see

### Key Metrics

A single metric card at the top:

- **STAKEHOLDERS** — the number of people associated with this grant series

### Tabs

The grant series detail page has 5 tabs:

#### Stakeholders

Lists all people associated with this grant series in a searchable, sortable table:

| Column | Description |
|--------|-------------|
| **Name** | Stakeholder's name with avatar |
| **Country** | Country flag and code (e.g., "CA, US") |
| **Profile** | Eligibility and accreditation labels |
| **Status** | Verification progress icons (hover for details) |
| **Signup** | Date and time the stakeholder signed up |

- **Filter by** status (All, Verified, Pending, etc.)
- **Sort by** signup date (ascending or descending)

#### Grants

The primary tab, showing all individual grants and pending invitations in a single table:

| Column | Description |
|--------|-------------|
| **CREATED ON** | Date and time the grant was created |
| **NAME** | Recipient's name with avatar |
| **VESTED** | Circular progress indicator showing vesting percentage (e.g., 100% filled blue circle) |
| **EXERCISED** | Circular progress indicator showing exercise percentage |
| **QUANTITY** | Number of units granted |
| **STATUS** | Current grant status (e.g., New, Vesting, Vested, Canceled) |

Click any grant row (or the arrow on the right) to open its detail page.

- **Search** by name or email
- **Filter by** status (e.g., All equity grants, Vesting, Vested, Exercised, Canceled)
- **Pagination** with 100 rows per page by default

Pending invitation links (not yet accepted) appear in the table with a **New** status label and a delete icon.

#### Invitation Links

Displays the "Grant Invitation Link" table with columns:

| Column | Description |
|--------|-------------|
| **INVITATION LINK** | Recipient name, email, and invitation URL with a copy icon |
| **QUANTITY** | Number of units assigned to this invitation |
| **ID VERIFICATION** | Whether identity verification is required (checkmark or X) |
| **TAX FORM** | Whether a tax form (W-9/W-8BEN) is required (checkmark or X) |
| **ACTIVE** | Toggle to enable/disable the invitation link |

#### Activity

A chronological log of events for this grant series, displayed in a table with columns:

| Column | Description |
|--------|-------------|
| **EVENT** | Description of the event with emoji prefix and timestamp (e.g., "🎊 The grant of 500 RSU towards Lala Doe is fully vested.") |
| **IP ADDRESS** | IP address of the user who triggered the event |

Events include: invitation link created, stakeholder signed up, grant accepted and signed, grant confirmed, grant fully vested, grant canceled, and more.

#### Terms

Displays the "Grant Terms" card with:

| Field | Example |
|-------|---------|
| **Vesting** | "2 months w/ 1 month cliff period" |
| **Compensation Type** | "RSUs" (or Options, Options ISO, Options NSO) |
| **Instrument** | Link to the compensation agreement PDF (e.g., "Fairbnb Dev – 1Mvesting1Mcliff Series – Compensation Agreement hellosign.pdf") |

For Options series, an **Exercise Price** field also appears.

Terms cannot be edited after the series is published.

## Individual Grant Detail Page

Click any grant in the Grants tab to see its detail page. This page has:

### Header

Shows the recipient's name, avatar, country, identity verification status, and a link to their full stakeholder profile.

### Grant Details

| Field | Description |
|-------|-------------|
| **Quantity** | Number of units granted |
| **Value** | Dollar value (shown only for token-issuing series) |
| **Compensation agreement** | Link to download the signed agreement, or status if unsigned |
| **Accepted on** | Date the recipient accepted the grant |
| **Tax form** | W-9 (US) or W-8BEN (international) if applicable |
| **Cliff period** | Duration and end date of the cliff |
| **Vesting period** | Duration and end date of vesting |
| **Expiration Date** | When the grant expires (if set) |
| **Invitation link used** | The invitation link the recipient used |
| **Exercise Status** | For exercised grants, links to the exercise investment |

### Contact Information

Shows the recipient's contact information.

### Status

Shows the grant's current status, a description of what it means, and action buttons for your next step (e.g., cancel, stop, issue tokens).

### Vesting Progress

Shown only for grants with a vesting schedule. Displays a visual indicator of how far along vesting has progressed.

## Grant Lifecycle & Statuses

Grants move through these statuses:

| Status | Color | Meaning | Available Actions |
|--------|-------|---------|-------------------|
| **PENDING SIGNATURE** | Blue | Recipient needs to sign the agreement | Cancel grant |
| **PENDING TAX** | Yellow | Recipient needs to verify identity and sign | Cancel grant |
| **CLIFF PERIOD** | Yellow | Grant is in the initial cliff period | Cancel grant |
| **VESTING** | Yellow | Grant is actively vesting | Stop grant |
| **VESTED** | Green | Grant is fully vested | — |
| **READY TO EXERCISE** | Yellow | Options are fully vested and ready to exercise | Exercise options |
| **READY TO ISSUE** | Yellow | Grant is vested and tokens can be issued | Issue tokens |
| **ISSUED** | Green | Tokens have been issued | See transaction on the blockchain explorer |
| **EXERCISED** | Green | Options have been exercised | — |
| **CANCELED** | Grey | Grant was canceled | — |
| **USER DECLINED** | Red | Recipient declined the grant | — |
| **AWAITING SIGNATURE** | Yellow | Signed by recipient, awaiting remaining signers | — |

### Multiple Signers

If your grant agreement requires more than one signature (e.g., a company officer needs to countersign), the grant stays in **Awaiting Signature** until all parties have signed. Each signer has their own signature card on the grant detail page — the **Sign** button only appears when you're logged in with the matching email address. Once everyone has signed, Fairmint saves the completed agreement and emails a copy to all parties.

### Cancel vs Stop a Grant

- **Cancel grant** — available during Pending Signature, Pending Tax, and Cliff Period. Completely cancels the grant before any equity has vested.
- **Stop grant** — available during Vesting. Stops future vesting while preserving what has already vested.

## Grants Overview Page

If you navigate to the top-level **Grants** section in the left menu (not a specific series), you see a consolidated view of **all grants across all series**. This page shows the same Grants table format but pulls from every grant series in your portal.
