# Cap Table Summary View

*This article is for founders and admins using Fairmint Studio.*

The Summary View shows your company's capitalization structure in a tabular format — who owns what, across all share classes and convertible securities.

Go to **Cap Table > Summary View** in the left menu, or click **Cap Table** directly (Summary View is the default).

For an overview of how to build your cap table, see [Import Your Cap Table](3.1-upload-import-cap-table.md).

---

## Before Your Cap Table Is Ready

If your cap table hasn't been set up yet, you'll see one of these states:

- **"Set up your cap table"** — your cap table hasn't been onboarded yet. Go to [Timeline View](3.6-cap-table-timeline-view.md) to upload documents.
- **"Cap table not ready"** — your cap table has been created but has no data yet. Upload more documents or wait for AI processing to complete.
- **Processing** — documents are still being processed. Check back shortly.

---

## Validation Warnings

If the system detects data issues in your cap table (e.g., invalid stock plans, missing vesting terms), a yellow warning banner appears at the top: **"We detected data issues in your cap table"** with a list of specific issues. These are informational — your cap table still displays, but you may want to fix the issues in [Timeline View](3.6-cap-table-timeline-view.md).

## Toolbar

When your cap table has data, the page heading shows **"Capitalization Table"** with a green **"OCF Compatible"** label (click it to learn more about the Open Cap Table Format). A toolbar on the right includes:

### Today / As-Of Date

Click **Today** to open the **As-Of date picker**, which lets you view your cap table as it existed on any past date. When active:

- The button changes to show the selected date (e.g., "Mar 13, 2026")
- A banner appears below the toolbar showing the historical snapshot context
- Click **X** next to the date to exit As-Of mode and return to the current view

> **Note**: As-Of mode and Pro-Forma mode are mutually exclusive — you can only use one at a time.

### Pro-Forma

Click **Pro-Forma** to model hypothetical scenarios on top of your current cap table. The **Pro-Forma Scenario** form lets you choose from three scenario types:

- **Financing Round** — model a new equity round (enter pre-money valuation, investment amount, stock class name, lead investor, and optional option pool expansion)
- **Convert SAFEs/Notes** — model conversion of outstanding convertible securities
- **Option Pool** — model an option pool expansion

When a Pro-Forma is active:

- The button turns orange and changes to **"Edit Pro-Forma"**
- A banner appears below the toolbar showing the scenario details
- Click **X** next to the button to exit Pro-Forma mode

Pro-Forma results are reflected in both Summary View and [Holder View](3.5-cap-table-holder-view.md), and included in exports.

### Export

Click **Export** to download your cap table as an Excel spreadsheet. The export includes multiple sheets:

- **Summary** — capitalization by share class (authorized, outstanding, fully diluted, liquidation, voting)
- **Holder View** — each stakeholder with their total holdings and percentages
- **Stock Ledger** — individual stock issuance records with dates, quantities, and prices
- **Stock Plan Ledger** — equity compensation grants with vesting details
- **Transaction History** — chronological record of all equity transactions
- **Context** — company metadata (name, formation date, authorized shares)
- **Legends** — stock legend templates and restrictions

If Pro-Forma mode is active, the export includes comparison sheets.

## Summary Capitalization Tab

The default tab shows a **Summary Capitalization** table with columns:

| Column | Description |
|--------|-------------|
| SHARE CLASS | Name of the share class (e.g., Common Stock) |
| SHARES AUTHORIZED/RESERVED | Total shares authorized for this class |
| OUTSTANDING SHARES | Shares currently issued and outstanding |
| FULLY DILUTED SHARES | Outstanding + all convertible/option shares |
| % FULLY DILUTED | Percentage of total fully diluted shares |
| LIQUIDATION | Liquidation preference amount |
| VOTING POWER | Number of votes this class controls |
| VOTING % | Percentage of total voting power |

Each share class appears as a row. If you have a **Stock Plan** (e.g., for equity compensation), it appears as a separate section with an "Available for Grants" sub-row showing the unallocated pool.

A **Total** row at the bottom sums all classes.

## Outstanding Convertible Securities Tab

If you have SAFEs, convertible notes, or other convertible instruments, a second tab appears: **Outstanding Convertible Securities**. It shows:

- Each convertible security with its terms (valuation cap, discount, etc.)
- The stakeholder who holds it
- Totals for outstanding convertible principal
